# Configuración ambiente virtual de python con `pipenv`

Documentación oficial [pipenv](https://pipenv-es.readthedocs.io/es/latest/)


#### Instalacion `pipenv`

```
pip install pipenv
```


### Instalar nuevas librerias
Ejemplo con `pandas`

```
pipenv install pandas
```

### Activar ambiente virtual

```
pipenv shell
```
